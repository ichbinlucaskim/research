# research

Interview prep repository for Google Cloud AI Agent Infrastructure — covering MCP, A2A, ADK, and multi-agent systems research.

## Structure

```
research/
├── mcp/        # Model Context Protocol — spec reference (PDF)
├── a2a/        # Agent2Agent Protocol — spec reference (PDF)
├── adk/        # Google Agent Development Kit — reference (PDF)
└── papers/     # Research papers with study guides and coding exercises
```

## Reading Order

1. `mcp/` — understand the tool-call protocol before anything else
2. `a2a/` — understand how agents communicate across trust boundaries
3. `adk/` — see how Google's framework implements both
4. `papers/` — read in order: 01 → 02 → 03 → 04 → 05 → 06 → 07 → 08
