# Papers

Reading list for Google Cloud AI Agent Infrastructure interview preparation.
All papers connect directly to MCP, A2A, and ADK.

## How to Use This Folder

1. Open the paper's `study_guide.md` first — read "The Problem" and "The Core Idea" sections before touching the PDF
2. Read the paper using the 5-step method (see root README)
3. Fill in the "Three-Line Note" in `study_guide.md`
4. Attempt `exercise_problem.py` for at least 20 minutes before looking at the solution
5. Check off items in the completion checklist
